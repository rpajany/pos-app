4. Deployment Steps
Local Testing:
Open terminal in your root folder.

Run: docker-compose up --build

Your app will be live at http://localhost:5000.

Deploying to Render via Docker:
Render makes this very simple:

Modify Service: Go to your Render Dashboard for pos-app.

Settings: Change the Runtime from Node to Docker.

Environment Variables: Ensure your MONGO_URI and JWT_SECRET are set in the Render "Environment" tab.

Deploy: Render will automatically detect the Dockerfile, build it, and launch it.

Why it didn't update automatically
Think of Docker like Git:

docker-compose build : is like a Commit (it saves changes locally).

docker-compose push  : is like a Push (it sends changes to the cloud/server).
